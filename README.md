# Offline Document Environment with Local AI Assistant

Этот проект представляет собой полностью автономную оффлайн-среду для хранения, поиска и работы с документами с помощью локального ИИ-ассистента на базе LLM Qwen-2.5-Coder и СУБД PostgreSQL (с расширением pgvector).

## Структура проекта

- `docker-compose.yml` - Конфигурация для запуска PostgreSQL с pgvector.
- `backend/` - Серверная часть на FastAPI (парсинг файлов, логика RAG, интеграция с Ollama и Postgres).
- `frontend/` - Веб-интерфейс на React (интерфейс чата и управления документами).
- `scripts/` - Скрипты резервного копирования, тестирования и обслуживания.
